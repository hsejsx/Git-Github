# 현재 폴더에 clone하기

- 그냥 `git clone 주소`를 입력하면 새 폴더가 생기지만, 현재 폴더에 바로 clone하고 싶다면 맨 마지막에 `.` 하나만 붙여주면 된다.

```
git clone 주소 .
```
